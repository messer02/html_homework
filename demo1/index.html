Введение в HTML и CSS

Это лекция о том, как начать работу с HTML и CSS, и как они помогают создавать веб-страницы.

Основы HTML

HTML (HyperText Markup Language) — это основной язык для создания веб-страниц. Он описывает структуру веб-контента с помощью тегов.

Теги HTML

Теги — это специальные инструкции браузеру, как отображать тот или иной контент. Тег имеет следующую структуру:

<tag>Контент</tag>

Например:

<p>Это абзац текста</p>

Теги могут быть парными (с открывающим и закрывающим тегами) или самозакрывающимися, как, например, <img>.

Основные теги HTML

  • Заголовки: Теги <h1> - <h6> создают заголовки разного уровня. Пример: 
    • <h1>Заголовок 1 уровня</h1>
<h2>Заголовок 2 уровня</h2>
  • Абзацы: Используются теги <p> для создания абзацев. Пример:
    • <p>Это абзац текста</p>
  • Ссылки: Тег <a> создаёт гиперссылки. Пример:
    • <a href="https://example.com">Перейти на сайт</a>
  • Изображения: Тег <img> вставляет изображения на страницу. Пример:
    • <img src="image.jpg" alt="Описание изображения">

Основы CSS

CSS (Cascading Style Sheets) отвечает за внешний вид веб-страниц. С помощью CSS можно задать цвета, шрифты, размеры, отступы и другие свойства элементов HTML.

Как подключить CSS

CSS можно подключить тремя способами:

  • Инлайновые стили: Используются внутри тега HTML. Пример:
    • <p style="color: red;">Этот текст будет красным</p>
  • Встроенные стили: Описываются в блоке <style> внутри HTML-документа. Пример: 
    • <style>
p {
  color: blue;
}
</style>
  • Внешний файл стилей: Стили хранятся в отдельном CSS-файле и подключаются через тег <link>. Пример: 
    • <link rel="stylesheet" href="styles.css">

Основные CSS-свойства

CSS предоставляет множество свойств для управления внешним видом элементов. Вот основные из них:

  • Цвета: Свойства color и background-color отвечают за цвет текста и фона. Пример: 
    • h1 {
  color: #ff6347;
  background-color: #f0f0f0;
}
  • Шрифты: Свойства font-family и font-size задают шрифт и его размер. Пример: 
    • p {
  font-family: Arial, sans-serif;
  font-size: 16px;
}
  • Отступы: Свойства margin и padding управляют внешними и внутренними отступами элемента. Пример: 
    • div {
  margin: 20px;
  padding: 10px;
}

Адаптивность и Flexbox

Адаптивность — это способность веб-страницы корректно отображаться на разных устройствах с различными размерами экрана. Flexbox — мощный инструмент для создания гибких макетов.

Использование Flexbox

Flexbox позволяет легко выравнивать элементы и управлять их размером в контейнере. Пример использования Flexbox:

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
